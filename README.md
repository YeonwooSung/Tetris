# Tetris

## 1. Overview

This is a tetris game written by C.

## 2. Usage

### 2 - 1. How to compile
To compile, use:

    make

It will output a binary file Tetris.


### 2 - 2. How to control

Controls:

    a: 		left
    s: 		bottom (quicker)
    d: 		right
    [space]: 	rotate
